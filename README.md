# Hacksher
This is a very nice hash bruteforcer written in Go, actually can break MD5, SHA1, SHA256 &amp; SHA512 (for now)

## Usage
the usage is very simple, you just need to do
```
./hacksher <hashType> <hash> <wordlist>
```
there you go.
